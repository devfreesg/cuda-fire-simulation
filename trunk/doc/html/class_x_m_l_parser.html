Parameters:
attributeName name of the node to find the floats. Node must be a child of the current root node
result array specifying the location to store the attribute's value
Returns:
true if successful, false otherwise
Note:
Given the xml data: 
 <xLowerBound>
   <x>-1</x>
   <y>-1</y>
 </xLowerBound>
with code: 
 float floatArray[2]; 
 getFloat2("xLowerBound",floatArray)
floatArray will become populated as {-1,-1}

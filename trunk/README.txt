Objective

Uses NVIDIA's CUDA architecture to implement a fire simulation application based on the 09 SIGGRAPH paper Directable, High-Resolution Simulation of Fire on the GPU by Christopher Horvath and Willi Geiger.
Background

This project was originally created for the Stanford classes CS 448X Math and Computer Science Behind Special Effects and CS 193G Programming Massively Parallel Processors with CUDA.
